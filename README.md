# IPL-Cricket-Analytics-Dashboard
Interactive Power BI dashboard analyzing IPL data (2008–2024) using Power BI, DAX, and Power Query.

## About the Project
The Indian Premier League generates a large volume of match and player data every season. This project leverages Power BI to clean, transform, and visualize the data, enabling users to analyze team performance, player statistics, and season-wise trends through an intuitive dashboard.


## Project Objectives
- Analyze team and player performance across IPL seasons.
- Monitor tournament KPIs using interactive dashboards.
- Identify season-wise trends and match outcomes.
- Deliver actionable insights through effective data visualization.


## Technical Skills Used
- Power BI
- Power Query
- DAX
- Microsoft Excel


## Dashboard Highlights
- Overall Tournament KPIs
- Team Performance Analysis
- Player Performance Analysis
- Orange Cap & Purple Cap Statistics
- Season-wise Performance Trends
- Interactive Filters & Slicers
- Drill-through Reports


## Key Insights
-  Analyzed **15+ IPL seasons (2008–2024)**.
-  Compared team performance across multiple seasons.
-  Evaluated player statistics, including runs, wickets, strike rate, and economy.
-  Identified winning trends and season-wise performance patterns.
-  Built KPI dashboards for quick and interactive analysis.
